## Optionals

# Frameworks

## Foundation
To use foundation, install node package:
```
npm install foundation-sites --save-dev
```

Uncomment foundation lines in:
```
app/app.init.js
app/app.scss
gulpfile.js
```

# Run

Development
```
npm run dev
```

Production
```
npm run production
```
